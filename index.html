Day Planner

Technologies: CSS 3 (Bootstrap), HTML 5, Javascript (jQuery, moment.js). I created a day planner that allows the user to say what they have done in the past, what they plan to do in the future, and what they are doing presently. The past, present and future time blocks all have designated color codes that will change according to the current time. This is able to be executed using moment.js. At the end the user can save what their day looked like in local storage.
