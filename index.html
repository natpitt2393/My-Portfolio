Work

Password Generator

Technologies: Javascript, HTML 5, CSS 3. This project allows the user to create a randomly generated password based on whether the user chooses to use special characters, numbers, and uppercase and lowercase letters.

A Quiz About the Sopranos

Technologies: Javascript, HTML 5, CSS 3. I created a quiz that is about New Jersey's favorite infamous crime family: the Sopranos. The user is asked to answer five questions. If they are incorrect ten seconds are deducted from the timer. The end score is equal to how much time is left on the clock. At the end the user can save their initials associated with their score.

Day Planner

Technologies used: CSS 3 (Bootstrap), HTML 5, Javascript (jQuery, moment.js). I created a day planner that allows the user to say what they have done in the past, what they plan to do in the future, and what they are doing presently. The past, present and future time blocks all have designated color codes that will change according to the current time. This is able to be executed using moment.js. At the end the user can save what their day looked like in local storage.
