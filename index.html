Work: Click on Title for Github link, click on image for deployed site!

Mental Health Tracker

Technologies: Javascript, HTML 5, CSS 3, MySQL. An app that allows you to create diary entries with different choices for mood input. After creating different diary entries you can see your mental health progress over time!

Everyone's A Critic

Technologies: Javascript, HTML 5, CSS 3, Bulma. A website that allows movie buffs and amateurs alike to look up movies they love or want to see. After finding some bare bone facts about the movie, the user can then decide to look up the New York Times review of that movie.

Day Planner

Technologies: CSS 3 (Bootstrap), HTML 5, Javascript (jQuery, moment.js). This website enables users to keep track of their busy lives by entering in descriptions for what they have to do each hour. Time slots are color coded and each note for planner can be saved.
